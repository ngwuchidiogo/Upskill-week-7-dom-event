# Upskill Week 7 - DOM & Events

This repo contains my submission for Week 7 of the Upskill Web Development Program.

## Tasks

### Task 1: DOM Explorer
- Select elements and log them to console
- Change heading text on page load
- Add new paragraph on button click

### Task 2: Interactive To-Do List
- Add tasks to a list
- Toggle task as completed
- Delete tasks
- Track total task count using an array

### Task 3: Event Propagation Demo
- Demonstrates event bubbling
- Uses stopPropagation() on button click

## How to Run
Just open `task1.html`, `task2.html`, or `task3.html` in a browser.
Check the console for Task 1 and Task 3.
